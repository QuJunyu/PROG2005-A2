Welcome to Inventory Management System

Application Purpose

This system is a lightweight, serverless inventory management solution developed with Angular and TypeScript, designed for retailers such as Harvey Norman. It enables efficient management of product inventory with core CRUD operations, popular item filtering, and name/category search functionality.

All data is stored session-persistent (lost on browser close) with no server-side interaction, ensuring local data security and fast operation. The system is fully responsive, supporting both mobile and desktop devices with an intuitive user interface.
